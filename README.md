# 🍽️ AROGYAA Restaurant Website

A modern, responsive restaurant website built for **AROGYAA**, a South Indian restaurant located in Chidambaram. The website showcases the restaurant's menu, featured dishes, gallery, customer testimonials, contact information, and location in a clean and professional user interface.

---

## 📖 Project Overview

AROGYAA Restaurant Website is designed to provide customers with an engaging online experience. The website highlights the restaurant's authentic South Indian cuisine, popular dishes, customer reviews, and contact details while maintaining full responsiveness across desktop, tablet, and mobile devices.

---

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd arogyaa-restaurant-website
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
